# Customer-Personality-Analysis-using-Clustering
Proyek ini bertujuan untuk melakukan Customer Segmentation menggunakan algoritma Clustering (K-Means). Tujuannya adalah mengelompokkan pelanggan berdasarkan karakteristik dan perilaku mereka, sehingga perusahaan dapat memahami profil pelanggan, membuat strategi pemasaran yang lebih tepat sasaran, dan meningkatkan retensi pelanggan.
